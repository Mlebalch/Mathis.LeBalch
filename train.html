Jeu "Train" Numérique

2023 Équipe de 2 Interface Graphique MVC JavaFX

Description

Adaptation numérique d'un jeu de société de stratégie ferroviaire développée en Java avec JavaFX. Le projet met en œuvre une architecture MVC complète avec une interface graphique interactive. Les joueurs posent des rails, construisent des connexions entre villes et marquent des points en complétant des trajets.

Technologies utilisées

Java 17 JavaFX 20 Maven Pattern MVC

⬇️ Télécharger et lancer

⚠️ Ce jeu est une application Java Desktop (JavaFX). Il ne peut pas tourner directement dans le navigateur. Tu dois avoir Java 17+ et Maven installés sur ton PC.
Télécharger le code source (.zip, ~51 MB)

Comment lancer le projet :

  • 1Extraire le .zip dans un dossier de ton choix
  • 2Ouvrir le projet dans IntelliJ IDEA (recommandé) ou VS Code + Extension Java
  • 3IntelliJ détecte automatiquement le pom.xml Maven → laisser télécharger les dépendances
  • 4Lancer la classe principale : fr.umontpellier.iut.trainsJavaFX.TrainsIHM
  • 5 Ou en ligne de commande dans le dossier du projet :
    mvn javafx:run

Points clés du projet

🎮 Jeu Complet Système de tours, plateau, gestion des scores
🏗️ Architecture MVC Séparation claire Modèle / Vue / Contrôleur
🖼️ Interface JavaFX UI fluide et intuitive en FXML
👥 Multijoueur Local Tour par tour jusqu'à 4 joueurs
