La Nuit de l'Info 2024

Décembre 2024 Équipe de 5 24h non-stop 🥉 Médaille Bronze Green Dev

Description

La Nuit de l'Info est un concours national de développement web organisé chaque année, qui se déroule en une seule nuit (de la tombée du jour à l'aube). Notre équipe de 5 personnes a conçu et développé un site web complet en 24h, en répondant au thème imposé tout en intégrant des défis bonus optionnels. Nous avons remporté la Médaille de Bronze dans la catégorie Green Development pour notre approche axée sur la performance et l'éco-conception.

Technologies utilisées

PHP 8.2 JavaScript ES6 HTML5 / CSS3 MySQL Bootstrap 5 API REST
