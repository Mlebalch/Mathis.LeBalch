CapyNote

Oct 2023 – Jan 2024 Équipe de 4 Gestion des Poursuites d'Études PHP / MySQL

Description

CapyNote est une application web de gestion des dossiers de poursuite d'études développée pour simplifier le suivi administratif des étudiants en fin de cursus. L'application permet aux étudiants de soumettre leurs candidatures, aux responsables de les traiter et de générer des rapports de suivi statistiques. Le projet a été réalisé en équipe de 4 sur 3 mois avec une méthodologie Agile (sprints Trello, revues hebdomadaires, burndown chart).

Technologies utilisées

PHP 8.2 MySQL HTML5 / CSS3 JavaScript TailwindCSS Docker DataTables.js Chart.js

Fonctionnalités clés

📋 Gestion des dossiers Soumission, suivi et validation des candidatures étudiant
👤 Multi-Rôles Étudiant, responsable pédagogique, administrateur
📊 Tableaux de bord Charts.js, DataTables, exports PDF automatisés
🐳 Docker Environnement conteneurisé pour le déploiement
🔒 Sécurité RGPD Chiffrement, hachage des mots de passe, droit à l'oubli
🗄️ SQL avancé Indexation, vues matérialisées, procédures stockées
